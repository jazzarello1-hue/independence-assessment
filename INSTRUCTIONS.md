# Setup Instructions

## Initial Setup

### 1. Create GitHub Repository

1. Go to https://github.com/new
2. Repository name: `independence-assessment`
3. Description: "Monthly independence readiness assessment for teenagers"
4. Make it **Public** (required for GitHub Pages)
5. **Do not** initialize with README (you'll upload yours)
6. Click "Create repository"

### 2. Upload Files

1. On your new repo page, click "uploading an existing file"
2. Drag and drop ALL files from this project:
   - `index.html` (the assessment)
   - `README.md` (documentation)
   - `INSTRUCTIONS.md` (this file)
   - `monthly-tracker.md` (tracking template)
3. Commit message: "Initial commit - Monthly independence assessment"
4. Click "Commit changes"

### 3. Enable GitHub Pages

1. Go to repository **Settings** (top menu)
2. Click **Pages** in the left sidebar
3. Under "Source":
   - Branch: Select **main**
   - Folder: Select **/ (root)**
4. Click **Save**
5. Wait 1-2 minutes for deployment
6. Your site will be live at: `https://[your-username].github.io/independence-assessment/`

## Monthly Usage

### For Teenagers

1. Visit the live website
2. Complete all four sections honestly
3. Click "Calculate My Score"
4. Review results with parents
5. Note scores in monthly tracker

### For Parents

1. Schedule monthly assessment meeting
2. Review scores together
3. Discuss specific areas:
   - Celebrate improvements
   - Identify growth opportunities
   - Set concrete goals for next month
4. Update monthly tracker
5. Follow up on previous month's goals

## Customization

### Updating Questions

1. Edit `index.html` file on GitHub
2. Find the question section you want to modify
3. Click the pencil icon (✏️) to edit
4. Make your changes
5. Scroll down, add commit message
6. Click "Commit changes"
7. Changes go live automatically in 1-2 minutes

### Adjusting Costs

Edit the cost values in the "What Your Parents Invest" section to match your actual expenses.

## Troubleshooting

**Site not loading?**
- Wait 2-3 minutes after enabling Pages
- Check Settings > Pages to confirm it's enabled
- Verify repository is Public

**Need to reset?**
- Just refresh the page - nothing is saved permanently
- Each month is a fresh start

**Want to track history?**
- Use the included `monthly-tracker.md` file
- Create one file per child
- Keep in a separate private repository or local folder
